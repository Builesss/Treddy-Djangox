**Fecha:** 12 de Mayo de 2026
**Tema:** Desarrollo del Control de Acceso y Navegación SPA

Durante la jornada se avanzó significativamente en el front-end y la lógica de seguridad:

1. Se implementó la vista de inicio de sesión utilizando AuthenticationForm personalizado.
2. Se crearon los distintos tableros (dashboards) según el rol del usuario autenticado (Cliente, Vendedor, Administrador).
3. Se introdujo una arquitectura Single Page Application (SPA) utilizando Fetch API de JavaScript, logrando que el menú lateral cambie los bloques principales sin recargar la página.
4. Se implementó NProgress (librería local) para mostrar una barra de carga en el borde superior al navegar.
5. Se añadieron decoradores de seguridad @login_required para proteger las rutas internas del tablero.

Con esto logramos una experiencia de usuario (UX) extremadamente fluida y moderna, emulando los comportamientos de frameworks como React, pero utilizando únicamente Vanilla JS y el motor de plantillas de Django.
